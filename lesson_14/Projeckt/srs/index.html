Підключення шрифтів на сайт

Найпопулярніші 2 способи, через гугол, або файли на сайті.

1 Спосіб за допомогою гугл

https://fonts.google.com/ - на цьому сайті. Пояснювати як - немає сенсу.

2 спосіб. Завантажуємо файли на сайт

1. Створюємо папку з назвою "fonts"
2. Завантажуємо в ту папку всі необхідні шрифти (формат ttf)
3. в файлі стилів прописуємо:
@font-face {
font-family: "RalewayRegular";
src: url("../fonts/RalewayRegular.ttf") format("truetype");
font-style: normal;
font-weight: normal;
}

4. Для потрібного класу чи тега присвоюємо даний шрифт:
h1{
font-family: "RalewayRegular";
}
